Privacy Policy

Last updated: June 3, 2026 | Website: www.studysphere.it.com

Data Stored On Device

StudySphere stores student profile details, tasks, plans, notes, flashcards, grades, reminders, game progress, and login activity locally on the device so the app can work without requiring an account server.

Consent

Students are asked for consent before profile and activity data is saved. The app uses this data to personalize planning, analytics, reminders, and progress views.

Free Access

StudySphere does not process payments in the current Android Play Store build. All included student tools are free for now.

Admin Access

Public Android builds do not include local admin credentials. Production admin access must be handled through a secured backend with proper authentication and authorization.

User Choices

Users can clear app data from the Android system settings by clearing StudySphere storage. A production backend release should also provide account deletion and data export options.

Contact

For privacy and support requests, contact the StudySphere project owner at leveragex254@gmail.com or through the Play Console tester communication channel.
