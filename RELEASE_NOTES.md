# StudySphere 1.0.1 Closed Testing Release Notes

StudySphere is a student productivity hub for planning, focus sessions, revision, reminders, notes, games, and progress tracking.

## What's Included

- Student login with remembered sessions.
- Dashboard with notifications, tasks, mood check-in, exams, and analytics.
- Planner and Study Autopilot tools.
- Focus timer and study streak tracking.
- Notes, flashcards, grades, files, groups, and reports.
- Brain-break games including Tic Tac Toe, Block Blast, Focus Trail, Memory, and Anagrams.
- Sage assistant for in-app guidance.
- In-app privacy policy.

## Release Safety

- Android build disables admin access.
- Paid access has been removed for this release; all included student tools are free for now.
- Release build targets SDK 36.
- Release build uses R8 and resource shrinking.
